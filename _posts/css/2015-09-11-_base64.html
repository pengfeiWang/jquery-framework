_base64.scss

path: base64-file/base64.scss

_base64 主要为了 兼容IE8加载一些GIF, 以及半透明文件
IE8下使用滤镜做半透明的话, 把透明层的其他内容也会半透明, 所以还是建议使用半透明文件
另外还有为了兼容IE9 let, css3动画制作的loading图标在IE9下也是不支持的, 这个文件也加载了一个不怎么美观的GIF loading图标

{% highlight javascript %} /* base64 提供的一些 scss 变量 */ /* loading gif图标 */ $loading /* 以下分别为对应的半透明 */ $opacity0 $opacity20 $opacity30 $opacity40 $opacity50 $opacity60 $opacity70 $opacity80 {% endhighlight %}
