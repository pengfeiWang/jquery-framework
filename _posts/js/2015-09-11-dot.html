doT js模板引擎

path: lib/dot/1.0.3/doT.js

号称效率最高的, 体积最小的js模板引擎

官方地址 doT

jekyll 模板与doT有冲突, 需要把 doT边界符转义输出
{% raw %} doT默认边界符"{{ ... }}", 本模块中边界符改成了 "{% ... %}" {% endraw %}

{% highlight javascript %} {% raw %} seajs.use('doT', function ( doT ) { var data = { name: '老王', docs: 'http://framework.note12.com' } var template = '姓名: {%=it.name%} 文档地址: 文档地址'; console.log( doT(template, data) ); /* 输出; 姓名: 老王 文档地址: 文档地址 */ }); {% endraw %} {% endhighlight %} {% raw %} {% endraw %}
