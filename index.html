前端说明文档

CSS

css依托于sass开发, 生产环境需使用编译后的css文件, sass不适用生产环境

编译 sass 建议使用koala, 更多关于软件信息移步koala

  1. scss源文件位于style/

JS

js多数依赖jQuery, 以jQuery插件形式存在

同时也支持 requirejs、seajs, 以及传统的 script标签按顺序引用

  1. require.js 位于 requirejs/
  2. sea.js 位于 seajs/
  3. 非依赖jquery的js文件位于 lib/modeleName/
  4. 依赖jquery插件位于lib/jquery-plugins/
